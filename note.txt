### connect wifi via nmcli
```
nmcli device wifi connect <SSID> password <PASSKEY> ifname <INTERFACE>
```

### install yay
```terminal
sudo pacman -S --needed base-devel git
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
cd ..
rm -fr yay
```

### install japanese
uncomment locale for japanese
```diff
- #ja_JP.UTF-8 UTF-8
+ ja_JP.UTF-8 UTF-8
```
then re-generate locale
```terminal
sudo locale-gen
```
install fonts
```terminal
sudo pacman -S noto-fonts-cjk
```

install fcitx5
```terminal
sudo pacman -S fcitx5-im fcitx5-mozc
```

add env to xprofile
```terminal
echo "GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx
" >> $HOME/.xprofile
```

now you may run fcitx5 its self background
and open fcitx5-configtool.
in fcitx5-configtool find mozc fron right pane
and move to left pane witch your Current Input Method.
finally you need to re-login to reload configuration you've done.
